# Attendance System Project

This is my RFID + ESP32 based attendance system made for college.

## Features
- ESP32 reads RFID card
- Shows attendance in serial monitor
- Buzzer gives feedback
- Sends data to Excel 

## Hardware Used
- ESP32
- RFID RC522 module
- Buzzer
- Jumper wires

## How to run
1. Open the code in Arduino IDE
2. Select ESP32 board
3. Upload the code
